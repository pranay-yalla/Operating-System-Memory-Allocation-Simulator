# Memory Allocation Simulator (C)

## Overview
This project simulates memory allocation techniques used in operating systems. It demonstrates how processes are allocated to memory blocks using different allocation strategies and compares their performance based on memory fragmentation.

## Algorithms Implemented
- First Fit
- Best Fit
- Worst Fit

## Features
- User input for memory blocks and processes
- Allocation table showing process allocation
- Fragmentation calculation after allocation
- Total fragmentation calculation
- Algorithm comparison to determine the most efficient strategy

## Concepts Used
- Operating Systems Memory Management
- Internal Fragmentation
- Memory Allocation Algorithms
- Arrays and Functions in C
